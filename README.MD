## 2. Quickly build local web player for h265 video.

### Start server

* python server
    ```bash
    bash start_server_py.sh
    ```

* node server
    ```bash
    bash start_server_js.sh
    ```

### Run Player
* Open the browser and visit `http://localhost:8999/demo.html`
